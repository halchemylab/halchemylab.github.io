Car Price Prediction

Machine Learning and Streamlit

Project Overview

This project applies exploratory data analysis (EDA), data cleaning, and machine learning to analyze customer data and predict car purchase amounts. The project concludes with the development of a Streamlit app, enabling interactive data exploration and real-time predictions. A Linear Regression model emerged as the best-performing predictive tool, demonstrating its capability to uncover actionable insights and support data-driven decisions.

Key Steps and Methodology

1. Exploratory Data Analysis (EDA)

  • Gender Distribution: Nearly equal distribution of male and female customers.
  • Age Ranges: Most customers aged 40–50, followed by 50–60.
  • Spending Trends: Males in the 40–50 age group had the highest purchase amounts, closely followed by females.
  • Correlation Analysis: Strongest correlations with Age (0.63), Annual Salary (0.62), and Net Worth (0.49).

2. Data Cleaning

  • No missing values required imputation.
  • Encoded Gender (0 = Female, 1 = Male).
  • No extreme outliers; scaled numerical features for model performance.

3. Feature Engineering

  • Created age range categories (e.g., 20–30, 30–40).
  • Aggregated spending metrics by gender and age groups.

4. Machine Learning

Linear Regression outperformed all models:

  • High R² Score.
  • Low RMSE.

The model was saved as a .pkl file for deployment.

5. Streamlit App Development

  • Data Exploration: Visualize trends by gender and age.
  • Predictive Insights: Input details to predict car purchase amounts.
  • Model Integration: Leveraged the Linear Regression model for real-time predictions.

Results and Impact

  • Insights: Understanding spending behavior aids marketing strategies.
  • Accuracy: Reliable predictions via robust machine learning techniques.
  • Accessibility: User-friendly Streamlit app for non-technical users.

Technical Stack

  • Data Analysis: Python (Pandas, NumPy, Matplotlib, Seaborn).
  • Machine Learning: Scikit-learn (Linear Regression).
  • App Deployment: Streamlit.
  • File Management: Pickle for model storage.

Key Takeaways

  • Conduct comprehensive EDA.
  • Engineer features for model performance.
  • Deploy intuitive apps for actionable solutions.
