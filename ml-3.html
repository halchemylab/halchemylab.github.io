Telecom Company Customer Churn Prediction Machine Learning Project

Machine Learning and Streamlit

Project Overview

The Customer Churn Prediction project focuses on identifying customers at risk of leaving a telecom service provider. Churn prediction is critical for businesses to proactively address customer retention challenges. The project leverages machine learning models to analyze customer behavior and predict the likelihood of churn. By providing a user-friendly interface for prediction, the application helps stakeholders make data-driven decisions to minimize churn and improve customer satisfaction.

Key Steps and Methodology

1. Exploratory Data Analysis (EDA)

  • Visualizing relationships between features like tenure, monthly charges, and churn status.
  • Analyzing demographic data, billing patterns, and their correlation with churn.
  • Identifying key drivers of churn through statistical summaries and visualizations.

2. Data Cleaning

  • Addressing missing or null values.
  • Removing duplicates and correcting inconsistencies.
  • Handling outliers to improve model robustness.

3. Feature Engineering

  • Categorical variables like gender are encoded into numerical formats..
  • Features such as tenure and monthly charges are scaled to ensure uniformity.
  • Derived features, like customer tenure categories, can provide additional insights.

4. Machine Learning

Logistic Regression and Random Forest were among the models evaluated:

  • High accuracy and F1-Score.
  • Balanced precision and recall for churn prediction.

The final model was saved as a .pkl file for deployment in the Streamlit application.

5. Streamlit App Development

  • Users can input age, tenure, gender, and monthly charges.
  • The app processes these inputs and provides a prediction ("Yes" or "No") with an engaging interface.
  • This simplifies churn prediction for non-technical stakeholders.

Results and Impact

  • Identifies high-risk customers, enabling targeted retention efforts.
  • Demonstrates measurable business value by reducing churn rates.
  • Empowers decision-makers with predictive analytics.

Technical Stack

  • Data Analysis: Python (Pandas, NumPy, Matplotlib, Seaborn).
  • Machine Learning: Scikit-learn.
  • App Deployment: Streamlit.
  • File Management: Pickle for model storage.

Key Takeaways

  • Customer Behavior Insights: Understanding patterns that drive churn.
  • Business Value: Enabling proactive measures to retain customers.
  • Technical Growth: Gaining expertise in machine learning, EDA, and app development.
